# InventoryPro
InventoryPro — небольшой проект состоящий из трех сущностей, написаный на языке Java с помощью Spring boot с использованием Graphql.
# Стек:
- Java 17
- Spring boot 3.3.0
- Graphql
- PostgreSQL
- Maven
- Liquibase
